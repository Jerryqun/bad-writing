## AI 算力产业链的传导逻辑

AI 芯片和算力基建的股价已经充分定价，真正的机会在于它们**溢出效应**带动的、尚未充分启动的衍生产业链。

```
已启动（高位）                正在传导（中位）             尚未充分启动（低位机会）
┌──────────────┐          ┌──────────────┐          ┌──────────────┐
│  AI 芯片      │          │  液冷散热     │          │  AI 电力基建  │
│  AI 服务器    │ ── 带动 →│  光模块/光通信 │ ── 带动 →│  具身智能/机器人│
│  算力基建     │          │  先进封装     │          │  AI 应用软件   │
│  PCB/HBM     │          │              │          │  AI 终端设备   │
└──────────────┘          └──────────────┘          │  AI 安全      │
                                                    │  AI 数据服务   │
                                                    └──────────────┘
```

> 以下梳理的公司均为**非创业板**（主板 / 中小板 / 科创板），数据更新于 2026 年 4 月。

## 一、AI 电力基建 ⚡（最确定的传导方向）

**传导逻辑**：AI 芯片越多 → 数据中心功耗爆炸（单机柜 120-150kW）→ 电力供应严重不足，从"缺芯"变成"**缺电**"。2026 年全球 AI 基础设施支出 4500 亿美元，其中电力成本占比超 40%。

## 二、具身智能 / 人形机器人 🤖（2026 量产元年）

**传导逻辑**：AI 大模型能力溢出到物理世界 → 机器人获得"大脑" → 人形机器人 2026 年进入**万台级量产**，2030 年市场空间有望达 210 万台。减速器、电机、传感器等零部件是最先受益的环节。

## 三、液冷散热 🌊（从"选配"到"标配"）

**传导逻辑**：AI 芯片功耗飙升 → 传统风冷触及物理极限 → 液冷成为数据中心**标配**。2026 年液冷数据中心渗透率有望从不到 10% 提升至 30%+。

| **佳力图** | 603912 | 主板 | 精密空调 + 液冷散热，数据中心温控 | 主板稀缺液冷标的 |

## 四、AI 应用软件 💻（从"卖铲子"到"用铲子"）

**传导逻辑**：算力基建完成后 → AI 大模型能力下沉 → 企业级 AI 应用/SaaS 迎来爆发。历史规律：互联网基建（光纤/交换机）→ 软件/互联网应用的传导周期约 2-3 年，当前正处于传导初期。这是**最后启动但空间最大**的方向。

## 八、投资注意事项

- **(1)** 产业链传导有**时间差**，过早介入可能面临长期横盘。建议结合 **MACD 金叉、KDJ 超卖区金叉** 等技术信号确认买点。
- **(2)** "尚未启动"不等于"一定会涨"，需要持续跟踪**订单数据、营收增速、行业政策**等基本面验证。
- **(3)** 科创板标的涨跌幅 **±20%**，波动高于主板和中小板（±10%），风险更高。
- **(4)** 关注**龙头公司的季报和订单公告**，业绩拐点是最可靠的启动信号。
- **(5)** 分散配置多个方向，避免单一赛道的集中风险。建议**核心仓位配电力基建 + 机器人**，卫星仓位配应用软件 + 终端设备。
