## 形状

```jsx
import './styles/three.css';
export default () => {
  return (
    <>
      <div class="sjx-1"></div>
      <div class="sjx-2"></div>
      <div class="box1 div"></div>
      <div class="box2 div"></div>
      <div class="box3 div"></div>
    </>
  );
};
```

## 使用 clip-path 实现图片切割成三角形

```jsx
import './styles/three.css';
export default () => {
  return (
    <>
      <div class="three-img" />
    </>
  );
};
```
