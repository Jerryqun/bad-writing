## 隐藏元素的 10 种方法

1、opacity:0 和 filter: opacity(0)  
2、display:none  
3、visibility:hidden  
4、rgba(x,x,x,0)  
5、transparent  
6、transform: scale(0);  
7、clip-path: circle(0);  
8、visibility: hidden  
隐藏元素但不改变文档的布局、 可以以捕获到事件  
9、z-index  
10、覆盖另一个元素
